## Câu Lệnh Điều Kiện

Java hỗ trợ nhiều loại câu lệnh điều kiện để kiểm tra điều kiện và thực thi mã tùy thuộc vào kết quả.

```java
int age = 20;
if (age >= 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}
